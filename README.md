# PrimoRPA-Custom-activities-and-examples

ChangingRange - Изменяет значения диапазона, работает с - и +.
Особенности: Предусмотрено несколько букв и сдвиг AZ+1 (Для использования ChangingRange нужно добавить в проект ChangingCell) 

CopyingRangeAnotherExcelDocument - копирует необходимый диапазон с 1 листа excel файла на другой лист другого excel файла (возможно использование для одного excel файла). (Точно работает только для формата: "A1", другой не проверялся)
Преимущества: вставляет все данные в форматирование с второго листа, нормально переносит объединенные ячейки. 
Важно: формулы переносятся, отображение не корректное

DividingCellIntoColumnAndNumber - делит значение ячейку на буквы и цифры, внезависимости от количества букв (Работает только для формата: "A1")

FindingOutermostFilledCells - получение диапазона с данными и измененными ячейками (если активность определение начальной и конечной строки работает не корректно + на выходе именно диапазон в формате "A1:A1")

Using a function in VBA - Пример использования VBA скрипта с выходным и входным значением

ChangingCell - Изменение ячейки с обработкой случаев AZ+1.

CopyingRangeAnotherExcelDocumentFormulas - копирует необходимый диапазон с 1 листа excel файла на другой лист другого excel файла (возможно использование для одного excel файла). (Точно работает только для формата: "A1", другой не проверялся)
Преимущества: вставляет все данные в форматирование с второго листа, нормально переносит объединенные ячейки. 
Важно: переносит формулы и они корректно отображаются

CustomTableSorting - На вход List c кастомным порядком сортировки без учета регистра, сортируемая таблица и название/номер колонки.
Преимущества: можно вводить не все значения, которые присутствуют в таблице, не введенные пойдут в конец. В List элементы должны идти по порядку, 
то есть {"1","3"} первыми значения будут "1", потом "3" и тд.
Можно вводить номер колонки или название, код обработает оба случая.
Применен алгоритм быстрой сортировки с улучшением для маленьких подмассивов, алгоритм взят остюда:
